# 🚀 TechMate

TechMate is an Android application that helps students connect with like-minded peers based on their tech interests (App Dev, Web Dev, AI/ML, etc.) to collaborate on projects and hackathons.

---

## ✨ Features

- 🔐 User Authentication (Login & Signup)
- 👤 Profile Screen with avatar and bio
- 📊 Stats section (Projects, Teams, Hackathons)
- 🧾 About section for user introduction
- ⚙️ Settings & Logout options

🚧 Upcoming:
- 📷 Profile image upload
- 🎯 Interest-based matching system
- ☁️ Firebase integration
- 🤝 Team collaboration features
